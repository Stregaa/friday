# Friday: Discord bot

*****
To create your own version of this bot:
1. Create an auth.json file in your bot folder
2. Create a bot on Discord website
3. Generate bot's token from Discord site
4. Add the following to auth.json: { "token": "" }, replacing empty quotations with your own generated token

*****
To run the bot:
1. Run [npm install] in your terminal
2. Run [node main.js] in your terminal
